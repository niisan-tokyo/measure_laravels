# LaravelS 計測

laravel + swooleでどれだけ早くなるか

# 条件一覧

## 環境条件

- laravels 単体
- Nginx + FPM
- apache
- ビルトインサーバ
- Nginx + laravels

## 計測条件

- 1000リクエスト
- 20 同時接続

# 計測結果

## 単純表示

以下のコマンドを使用

```
ab -c 20 -n 1000 http://localhost:8080/
```
